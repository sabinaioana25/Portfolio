Duration: 2 weeks

Tools used: Android Studio, Adobe XD & Photoshop

Baking App - Github Project

description

My task was to create a Android Baking App that will allow Udacity‚Äôs resident baker-in-chief, Miriam, to share her cake recipes. The app allows the user to select a recipe and see video-guided steps on how to make a dessert.

features

The user can open a detailed recipe by clicking on a dessert card and can jump to another recipe from whithin the detail screen.
More details about the recipe are available in the detail screen.

project requirements

- Project is written only in Java
- JDK 1.8
- Android SDK
- Android Nougat (API 25)
- Latest Android SDK Tools (Android studio 3.0 canary) and build tools

implementation

The project uses the following features:
- Exoplayer to display videos
- Home screen widget
- Third-party library
- Fragments to create a responsive design that works on phones and tablets
- UI testing (JUnit 4 and Espresso)

Data persistance

The app makes a request of the recipe data from the network and saves the data in the local storage as SharedPreferences. This data is then presented to the user in a RecyclerView.

UI

The UI is crafted following responsive and material design guidelines for phone and tablet screens.
