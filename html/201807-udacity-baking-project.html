Baking App

Mobile / UX

intro

This project comes under Udacity's Android Development Nanodegree. The scope of the project is to design and plan the steps necessary to create a production-ready application. The JSON file containing the recepies' instructions, ingredients, videos and images are provided by Udacity.

Objective: build an app of my own design in two stages (design & plan, code build)

Duration: 2 weeks

Tools used: Android Studio, Adobe XD & Photoshop

‚ÄçFinal product showcase: An app to view video recipes

description

My task was to create a Android Baking App that will allow Udacity‚Äôs resident baker-in-chief, Miriam, to share her recipes with the world. The app would allow a user to select a recipe and see video-guided steps for how to complete it.

features

Upon clicking a recipe, the user is presented with a list of steps. The user can also access the ingredient list. The app also has functionality to change the recipe within the steps list activity.
In this activity, the user can also save the recipe to a screen widget. Upon selecting a step, the user is presented with more details about the steps and also video tutorials (if provided in the JSON file).

project requirements

- Project is written using only Java
- JDK 1.8
- Android SDK
- Android Nougat (API 25)
- Latest Android SDK Tools (Android studio 3.0 canary) and build tools

implementation

The project uses the following functionality features:
- Exoplayer to display videos
- Home screen widget
- Third-party library
- Fragments to create a responsive design that works on phones and tablets
- UI testing (JUnit 4 and Espresso)

Data persistance

The android application makes a request of the recipe data from the network and saves the data in the local storage as SharedPreferences. This data is then presented to the user in a RecyclerView.

UI

The UI is using responsive design and material design guidelines, for phone and tablet screens.
