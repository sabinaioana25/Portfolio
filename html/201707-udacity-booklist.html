Mobile / UX

intro

This project comes under Udacity's Android Basics Nanodegree Program. The goal was to design and create the structure of a Book Listing app which would allow the user to get a list of published books on a given topic. The details on each book is taken from the Google Books api.

Objective: build a simple UI and display a list of books on any topic

Duration: 1 week

Tools used: Android Studio, Adobe XD

‚ÄçFinal product showcase: an app which would allow a user to get a list of published books on a given topic

features

With the app, the user can:
- search for books by name or title
- see a grid list of books
- access additional details on a book with a simple click (e.g. original title, launch year)
- read reviews of a selected movie

project requirements

- Project is written using only Java
- JDK 1.8
- Android SDK
- Android Nougat (API 25)
- Latest Android SDK Tools (Android studio 3.0 canary) and build tools

implementation

The project uses the following functionality features:
- fetch data from an API
- use an AsyncTask
- create a list based on that data and display it to the user
- use Fragments to display different lists, based on the user's preferences

UI

The UI is using responsive design and material design guidelines, for phone screens.
