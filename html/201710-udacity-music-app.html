Rare Beats

Mobile / UX

intro

This is a prototype UI for a music player application. It is the forth project under Udacity's Android Basics Nanodegree Program.

Objective: Design of a music application

Duration: 2 days

Tools used: Android Studio, Adobe XD, Adobe Photoshop

‚ÄçFinal product showcase: A prototype of a music app using intents and sub-activities


Rare Beats - Github Project

features

With the app, the user can:
- create an account
- pick a genre from a list
- search for an artist or band
- add track to a list of favorites

project requirements

- Project is written using only Java
- JDK 1.8
- Android SDK
- Android Nougat (API 25)
- Latest Android SDK Tools (Android studio 3.0 canary) and build tools

implementation

The project has the following features:
- a ListView for displaying artist, genres and albums
- click intents for navigating the app activities
